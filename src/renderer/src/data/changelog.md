# Agent Arena

**当前版本：v0.1.1**

养成 AI 角色、组织规则化社交对局的 Electron 桌面端。

---

## v0.1.1

### 修复与改进

- **初始素材**：远程完整素材包下载失败时，可跳过并使用内置默认素材完成初始化
- **素材回退**：素材包中缺少某角色立绘/头像时，自动使用「默认」占位图，不再阻断角色创建
- **初始化向导**：素材相关错误统一展示「跳过 / 导入 zip / 重试下载」选项

---

## v0.1.0

### 核心能力

- 大厅、角色库、玩法场景与对局记录
- Platform 账户体系、钱包充值与用量统计
- 模型网关对接：概览、连通性调试与流式测试
- Agent Labs 测评榜与设置中心整合

### 数据与安全

- 角色、对局与配置保存在本地数据库
- 模型请求经 Platform 网关，Key 仅存本机

### 后续计划

- 成就系统与用户统计扩展
- 帮助中心智能问答增强
